# Simple Demo starten (ohne Build-Setup)

Du kannst die Demo direkt als einfache HTML-Datei öffnen:

1. Datei `index.html` doppelklicken
2. Oder per kleinem lokalen Server starten:
   - `python3 -m http.server 8080`
   - dann `http://localhost:8080/index.html`

Es sind **keine npm-Installationen** nötig.
